# 📊 User Data Analysis Project

A simple project for analyzing user data using Python, aimed at extracting useful insights and visualizing them clearly.

---

## 🎯 Project Overview
This project:
- Calculates the average time spent
- Analyzes data based on gender
- Visualizes the results using a bar chart

---

## 🛠️ Tools Used
- 🐍 Python  
- 📊 Pandas  
- 📈 Matplotlib  

---

## 📁 Project Files
- analysis.py → main analysis script  
- data.csv → dataset  

---

## 🚀 How to Run
1. Install required libraries:
pip install pandas matplotlib

2. Run the script:
python analysis.py

---

## 📌 Output
- Prints the average time in the terminal  
- Displays a bar chart comparing categories
